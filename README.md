# Tekkie

A project built to display knowledge of acquiring data from API's. App shows information about crypto currencies and sources top news stories around tech.

## Built with
* Nextjs (React)
* Framer Motion
* TailwindCSS
* Coin Gecko API
* Hacker news API

## Features 
* Animations 
* Responsive design
* Filtering
